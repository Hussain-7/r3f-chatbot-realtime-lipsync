# r3f-chatbot-realtime-lipsync

A real-time 3D chatbot project with lip-sync animation, split into separate frontend and backend applications.

## Project Structure

- `frontend/` - React Three Fiber client for the animated character experience
- `backend/` - Express-based service for model responses and audio generation

## Highlights

- Realtime chatbot interaction
- Lip-sync and avatar animation workflow
- Separate frontend and backend development environments
- OpenAI and ElevenLabs integration on the backend

## Getting Started

### Frontend

```bash
cd frontend
yarn
yarn dev
```

### Backend

Create a `.env` file in `backend/` based on the expected API keys, then run:

```bash
cd backend
yarn
yarn dev
```
